# 🎮 GameVerse

Aplicação web de gerenciamento de jogos, construída com HTML, CSS e JavaScript puro, consumindo a API REST hospedada no Render.

---

## 📋 Objetivo

Permitir que o admin visualize, cadastre, edite e remova jogos por meio de uma interface simples e responsiva, por sua vez o cliente só pode visualizar os jogos, integrando-se ao backend disponibilizado para a disciplina.

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS / ES Modules)

---

## ✅ Funcionalidades Implementadas

- Listagem de todos os jogos cadastrados
- Cadastro de novo jogo
- Edição de jogo existente
- Remoção de jogo
